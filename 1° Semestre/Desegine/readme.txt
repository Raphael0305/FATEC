Atividade: Crie posts para redes sociais


Tema: datas comemorativas do seu dia de aniversário.​


https://www.calendarr.com/brasil/datas-comemorativas-2024/

1-Criar uma paleta de cores

2-A imagem deve ter pelo menos: título da live/projeto, a data, e um pequeno texto sobre a data.

3-Escolha formas geométricas que possam ser associadas ao tema. Formas devem ser criadas como imagem vetorial

4-Use uma foto para compor o post. A foto deve ser editada para apagar o fundo.

5-Escreva um texto breve justificando suas escolhas para a paleta de cores, da família tipográfica, das formas empregadas e da composição dos elementos. (No próprio Figma).

6-Adaptar o mesmo post para os seguintes formatos:
	a. Instagram Feed: 1080px x 1080px;
	b. Instagram Stories: 1080px x 1920px;
	c. Facebook, post com link e imagem: 628px x 1200px.

Link para o documento em Figma na tarefa. Não se esqueça de dar permissão para visualização para qualquer um com link.
